## Diploma document structure

1. Introduction
    1.1 Project Background (Сэдвийн үндэслэл)
    1.2 Problem Statement (Асуудлын тодорхойлолт)
    1.3 Project Objectives (Төслийн зорилго)
    1.4 Scope and Limitations (Хамрах хүрээ ба хязгаарлалт)
2. Literature Review (Related Work)
    2.1 Existing Systems/Studies (Ижил төстэй системүүд)
    2.2 Comparative Analysis (Харьцуулсан шинжилгээ - table)
    2.3 Proposed Solution Overview (Санал болгож буй шийдэл)
3. Requirements & Planning
    3.1 Functional Requirements (Функциональ шаардлага)
    3.2 Non-Functional Requirements (Функциональ бус шаардлага)
    3.3 Scope statement (Ажлын хүрээ)
    3.4 Development plan (Хөгжүүлэлтийн төлөвлөгөө)
4. Methodology & System Design
    4.1 System Architecture (Системийн архитектур)
    4.2 Database design (Өгөгдлийн сангийн зохиомж)
    4.3 Detailed design/Logic Flow (Нарийвчилсан дизайн ба логик дараалал)
    4.4 User Interface Design (Хэрэглэгчийн интерфэйс)
